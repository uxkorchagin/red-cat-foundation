# Platform / Components

Спецификация архитектуры компонентов Foundation: API, состояния, поведение, композиция — платформенно-независимо. Источник истины о структуре компонента, а не об обосновании решений (обоснования — в [knowledge/components/](../../knowledge/components/)).

Новые спецификации компонентов оформляются по [templates/component-spec.md](../../templates/component-spec.md) и проходят [checklists/component-review.md](../../checklists/component-review.md).
