# Platform / Tokens

Спецификация дизайн-токенов Foundation — платформенно-независимого визуального языка. Источник истины о значениях, а не об их обосновании (обоснования — в [knowledge/foundations/](../../knowledge/foundations/)).

Архитектура уровней токенов (global/alias/component), нейминг и категории — предмет проектирования, следующего за созданием этой структуры репозитория.

Новые спецификации токенов оформляются по [templates/token-spec.md](../../templates/token-spec.md).
