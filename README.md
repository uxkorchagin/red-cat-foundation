# Red Cat Foundation

Red Cat Foundation — внутренняя инженерная платформа студии Red Cat. Единый источник истины для проектирования цифровых продуктов: дизайн-токены, архитектура компонентов, документация, процессы.

Это не дизайн-система отдельного продукта. Это платформа, на основе которой стартует каждый новый проект студии. Полное обоснование — в [constitution/vision.md](constitution/vision.md).

## С чего начать

1. Прочитать [constitution/vision.md](constitution/vision.md) — зачем существует Foundation.
2. Прочитать [constitution/principles.md](constitution/principles.md) — архитектурные принципы, которым подчиняется любое решение в системе.
3. Свериться с [constitution/glossary.md](constitution/glossary.md) — единый словарь терминов.
4. Перед внесением изменений — изучить [governance/contribution-process.md](governance/contribution-process.md).

| Раздел | Назначение |
|---|---|
| [constitution/](constitution/) | Конституция Foundation — фундаментальные документы, которые меняются крайне редко: vision, principles, glossary, roadmap, исходный Project Requirements. |

| [releases/](releases/) | Материалы конкретных релизов Foundation. |

## Текущий этап

Foundation находится на Этапе 1 — построение технического фундамента (токены, архитектура компонентов). UX-стандарты и DesignOps-процессы — следующие этапы, см. [constitution/roadmap.md](constitution/roadmap.md).

## Источник истины

Весь репозиторий — единственный источник истины для проектирования цифровых продуктов студии. Дизайн-инструменты (Figma) дополняют его, но знания фиксируются здесь.
