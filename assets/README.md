# Assets

Вспомогательные материалы, используемые документацией и базой знаний: схемы, диаграммы, иллюстрации, изображения. Assets не содержат знания сами по себе — они используются `knowledge/`, `constitution/` и другими доменами для иллюстрации.

- [diagrams/](diagrams/) — архитектурные и процессные схемы.
- [illustrations/](illustrations/) — иллюстративные изображения.
