# Knowledge / Governance

Знания о том, почему процессы управления Foundation устроены именно так — в отличие от [governance/](../../governance/), где описан сам процесс (что делать), здесь фиксируется обоснование (почему именно так), альтернативы, которые были рассмотрены, и опыт применения процессов.

Наполняется по мере накопления практики использования RFC-процесса, версионирования и ревью.
