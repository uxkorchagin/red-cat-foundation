# Knowledge / Foundations

Обоснования и накопленный опыт, связанные с фундаментальным визуальным языком Foundation: дизайн-токены, сетки, spacing, breakpoints, типографика и другие атомарные принципы.

Здесь фиксируется **почему** Platform Core устроен именно так — в отличие от [platform/tokens/](../../platform/tokens/), где находится сама спецификация.

Наполняется по мере проектирования архитектуры токенов на Этапе 1.
