# Knowledge / Figma

Знания об архитектуре Figma: организация библиотек, работа с переменными, структура файлов, практики поддержки компонентов и токенов в Figma. Опирается на накопленный практический опыт студии.

Не является инструкцией по конкретным операциям в Figma — это обоснования и рекомендации, объясняющие принятые архитектурные решения.

Наполняется по мере того, как архитектура токенов и компонентов (Этап 1) начинает проецироваться на Figma-библиотеку Foundation.
