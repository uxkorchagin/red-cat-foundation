# Checklist: Release

Проверка готовности релиза перед публикацией версии Foundation. См. [governance/release-process.md](../governance/release-process.md).

- [ ] Версия классифицирована корректно (major/minor/patch) согласно [governance/versioning.md](../governance/versioning.md).
- [ ] Все включённые изменения прошли ревью ([governance/review-process.md](../governance/review-process.md)).
- [ ] Запись добавлена в [CHANGELOG.md](../CHANGELOG.md).
- [ ] Материалы релиза подготовлены в [releases/](../releases/).
- [ ] Для major-версии — подготовлена миграционная инструкция для команд проектов.
- [ ] Описано влияние на уже запущенные проекты.

## Кто проводит проверку

Lead Designer / Design Systems Architect, при поддержке Skill `governance-qa`.

## При несоответствии

Релиз не публикуется до устранения несоответствия.
