# Releases

Материалы, относящиеся к конкретным релизам Foundation: release notes, миграционные инструкции для major-версий. См. [governance/release-process.md](../governance/release-process.md).

В отличие от [CHANGELOG.md](../CHANGELOG.md) (краткая хронологическая запись в одну строку на изменение), здесь хранятся полные материалы конкретной версии.

Раздел пуст до выпуска первой версии Foundation.
